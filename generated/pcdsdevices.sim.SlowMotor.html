pcdsdevices.sim.SlowMotor¶

class pcdsdevices.sim.SlowMotor(*args, init_pos=0, **kwargs)¶

Simulated slow-moving motor.

Unlike the FastMotor, this takes some time to reach the destination. Use this when you need some sort of delay.
