pcdsdevices.mv_interface.FltMvInterface¶

class pcdsdevices.mv_interface.FltMvInterface(*args, **kwargs)¶

Extension of MvInterface for when the position is a float.

This lets us do more with the interface, such as relative moves.

presets¶

Presets – Manager for preset positions.
