pcdsdevices.sim.SlowMotor¶

class pcdsdevices.sim.SlowMotor(*args, **kwargs)¶

Simulated slow-moving motor.

Unlike the SynAxis built into ophyd, this takes some time to reach the destination. Use this when you need some sort of delay.
