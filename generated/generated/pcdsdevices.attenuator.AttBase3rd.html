pcdsdevices.attenuator.AttBase3rd¶

class pcdsdevices.attenuator.AttBase3rd(prefix, *, name, **kwargs)¶

Attenuator class to use the 3rd harmonic values.

This is exactly the same as the normal AttBase, but with the alternative transmission PVs. It can be instantiated using the use_3rd argument in the Attenuator factory function.
