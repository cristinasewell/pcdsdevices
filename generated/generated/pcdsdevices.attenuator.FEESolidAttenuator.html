pcdsdevices.attenuator.FEESolidAttenuator¶

class pcdsdevices.attenuator.FEESolidAttenuator(prefix='', *, name, kind=None, read_attrs=None, configuration_attrs=None, parent=None, **kwargs)¶

Solid attenuator variant from the LCLS-II XTES project.

Motorized, 18 filters + 1 inspection mirror.

This is a quick-and-dirty Ophyd device for controlling AT2L0 motion and generating a PyDM control screen.

Parameters

  • prefix (str) – Full Solid Attenuator base PV.

  • name (str) – Alias for the Solid Attenuator.
