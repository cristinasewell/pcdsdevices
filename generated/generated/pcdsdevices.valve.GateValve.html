pcdsdevices.valve.GateValve¶

class pcdsdevices.valve.GateValve(*args, **kwargs)¶

Basic Vacuum Valve

This inherits directly from Stopper but adds additional logic to check the state of the interlock before requesting motion. This is not a safety feature, just a notice to the operator.

This class has been replaced by VGCLegacy but has not been removed as some elements still need to be carried over.
