pcdsdevices.mv_interface.tweak_base¶

pcdsdevices.mv_interface.tweak_base(*args)¶

Base function to control motors with the arrow keys.

With one motor, this will use the left and right arrows for the axis and up and down arrows for scaling the step size. With two motors, this will use left and right for the first axis and up and down for the second axis, with shift+arrow used for scaling the step size. The q key quits, as does ctrl+c.
