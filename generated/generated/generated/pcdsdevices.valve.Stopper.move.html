pcdsdevices.valve.Stopper.move¶

Stopper.move(position, moved_cb=None, timeout=None, wait=False)¶

Move to the desired state and return completion information.

Parameters:
  • position (int or str) – The enumerate state or the corresponding integer
  • moved_cb (function, optional) – moved_cb(obj=self) will be called at the end of motion
  • timeout (int or float, optional) – Move timeout in seconds
  • wait (bool, optional) – If True, do not return until the motion has completed.
Returns:

status – Status object that represents the move’s progress.
Return type:

StateStatus
