pcdsdevices.valve.Stopper¶

class pcdsdevices.valve.Stopper(*args, **kwargs)¶

Controls Stopper

A base class for a device with two limits switches controlled via an external command PV. This fully encompasses the controls Stopper installations as well as un-interlocked GateValves

Parameters

  • prefix (str) – Full PPS Stopper PV

  • name (str) – Alias for the stopper

commands¶

An enum with integer values for open_valve, close_valve values
