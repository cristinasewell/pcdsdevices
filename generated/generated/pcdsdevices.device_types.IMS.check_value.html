pcdsdevices.device_types.IMS.check_value¶

IMS.check_value(value)¶

Raise an exception if the motor cannot move to value.

The full list of checks done in this method:

  • Check if the value is within the soft limits of the motor.
  • Check if the motor is disabled (.DISP field)`
  • Check if the spg field is on “pause” or “stop”
Raises:
  • MotorDisabledError – If the motor is passed any motion command when disabled, or if the .SPG field is not set to Go.
  • LimitError(ValueError) – When the provided value is outside the range of the low and high limits
