pcdsdevices.device_types.Stopper¶

class pcdsdevices.device_types.Stopper(*args, **kwargs)¶

Controls Stopper.

A base class for a device with two limits switches controlled via an external command PV. This fully encompasses the controls Stopper installations as well as un-interlocked GateValve s.

Parameters

  • prefix (str) – Full PPS Stopper PV.

  • name (str) – Alias for the stopper.

commands¶

An enum with integer values for ~Commands.open_valve and ~Commands.close_valve values.
